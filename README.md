## Montra Cloud - The cloud backup for my personal finance tracking application Montra
This is a cloud based backup for storing transactions made in my local-first mobile application [Montra] (https://github.com/aeron-cap/montra). It is personally tailored to my methods of finance management.

## Motivation
I created this app together with the local-first mobile Application for self-learning and to have the option to backup the transactions, etc. into the cloud. Editing and adding transactions in Google Sheets has been an inconvenient way of personal finance that I have endured, since I have been mostly making inputs through my phone and seldomy on my Laptop.

## Tech Stack
<b> Built With </b>
- [NestJS] (https://nestjs.com/)
- [Drizzle] (https://orm.drizzle.team/) (ORM)

## The Future
The intergration of this cloud backup to the mobile application is still ongoing, after that, there will be a web-based client for accessing the same data we can find on the mobile application.
